# HR 面试题

# 1. 你为什么学前端？

  1. 在前期的学习过程中，自己真正体验到了乐趣和成就感，一发不可收拾。
  2. 前端的技术一直推陈出新(3-6个月)，而且新技术出来后，几乎立马就能使用，不断学习，不断应用，这样的工作不会枯燥乏味。
  3. 前端处于软件开发流程中，位于中间的环节，会接触到很多使用不同技术的小伙伴(视觉，交互，后台)，我很喜欢与人打交道，向别人获取新知识，和大家分享自己的经验。

自己很喜欢做前端，因为前端做出来的东西可以马上展现出来，可以给我带来最直接的成就感，不像后端做的大部分都是数据层面的东西，不能让客户老板和自己马上看到。 而且这两年前端技术发展很快，功能越来越强大，所以做的时间越久，越感觉可挖掘的点很多，很有意思。

# 2. 职业规划？

一、起步阶段：

  1. 基础知识的掌握 除了理论上的知识外，我们在过程中更多的是需要投入到实践中去；
  2. 常用工具的掌握 例如：vscode，这种工具的掌握能够提升专业、职业度，同时能够提升工作效率；
  3. 保持良好的开发习惯 例如：在开发一个项目之前可以先分析、先写文档、适当的填写注释；

二、提升阶段：

  1. 高级技术的掌握 例如：学习一些算法、了解底层原理、看一些网站项目的源码等；
  2. 各种框架的学习 框架是我们提高工作效率的一种优秀手段；例如：vue、react、angular、jquery、flutter等；
  3. 各种网络协议的分析 众所周知 web 前端技术就是云上的技术，云的协议有很多种，可以着重学习：TCP/IP、UDP等协议；

三、成型阶段 去掌握 MVVM 框架的实现原理，精通 Webpack、glup等自动化部署的开发，编写高效的算法函数； 另外到了这个阶段说明已经有了 Web 资深前端工程师的水平了，当然我认为还是需要去新技术的研究，互联网技术发展速度日新月异，只有不断的去学习才能不被淘汰。

# 3. 你的优点 / 缺点是什么？

优点：
我的性格比较稳重，比较踏实，做事很认真，而且比较有计划性，每天做的第一件事就是列出当天的计划把当天要做的事情分成两种：必须完成的和最好能完成的。出去购物或者其他前都会列一个清单。还有一个优势是我的适应性很强。

缺点：
我的执行力不错，通常领导交代的任务我都能完成。但是正因为如此，我主动地深入思考就相对少了一些，更多是在行动上。如何站到更高的格局上去看待工作，如何思考战略布局，这类深度的专研我还需要加强。

# 4. 你有什么爱好？平时的闲暇时间你都干些什么？

喜欢长跑。在学院田径队待了三年，每年都会参加校运动会并取得了好成绩。平时喜欢逛 IT 论坛，比如 CSDN、B站、知乎、掘金等。利用业务时间学了前端开发。

# 5. 简述学生生涯中遇到的印象最深刻的困难以及解决方案？

我刚去公司实习时，我主要负责产品web端的测试工作，就是写测试用例，然后执行用例。我第一天去上班时 我的企业导师给我布置了一个作业，让我写出一个测试用例，当时不知道该怎么编写测试用例，然后我上网查阅资料，查看相关的一些案例，企业导师也很耐心地指导我。最后写完导师很满意，从此开始学会了写测试用例。

# 6. 你经历过最让你失望的事情是什么？

刚上大四时，就想找一个稳定的工作，能为企业终身服务。后来发现，找一个诚心如意的工作的确不容易。找到好的工作需要花费一些时间，但是我认为是值得的。

# 7. 你如何让别人接受你的观点和主意？

有时候我发现好的主意，甚至是问答的观点，并不能被人们所认同。有一点我很赞同，表达观点的方式同观点一样重要。我通常会从他人的角度来看待这个观点，这样的话就能很容易找到说服他们的方法。

# 8. 你如何对待别人的批评？

沉默是金，在大家情绪化的时候，不必说什么，否则情况更糟，不过我会接受建设性的批评。最后，我会等大家冷静下来再讨论，是人的问题，还是流程的问题，并且进行改正和优化。

# 9. 你如何对待自己的失败？

首先分析失败的原因，能力问题还是工作态度问题，能力不够可以学，摆正心态去学习追赶前辈，态度问题就要反省自己，立即改正。我们大家生来都不是十全十美的，我相信我有第二个机会改正我的错误。

# 10. 有句谚语说：“目标刻在岩石上，计划写在沙滩上。”你是怎么理解的？

写在石头上就意味着刻上去了不会再变动了，而写在沙滩上则很容易擦掉。
我的理解是：目标一旦确立就不会轻易改变，而计划就像是写在沙滩上的字儿一样可以很轻易地擦掉，就是说计划是随着情况不断改变的。不是有句话叫计划赶不上变化嘛，就是这个道理。

# 11. 你找工作时，最重要的考虑因素为何？

首先是岗位，然后是行业。选择一个行业，决定了我们未来十年二十年的工作方向。福利水平（五险一金等）、公司同事的氛围。

# 12. 你认为你在你的专业应聘上有什么劣势和不足？

我认为做为一名应届毕业生，我在专业上做掌握的更多来自大学期间的理论基础知识，缺乏实践操作，所以我最大的劣势和不足就是缺乏工作经验。

# 13. 你最讨厌什么工作岗位？

我没有讨厌的工作岗位。我认为一个国家里每份职业都有存在的价值，同样一个企业里每个岗位都是企业所需，都能创造劳动财富，在我看来，凡是付出劳动，创造成果的岗位都是值得尊重的岗位，我都不会讨厌。

# 14. 你对薪资的要求？

我对工资没有硬性要求，我相信贵公司在处理我的问题上会妥善合理。我注重的是找对工作机会，所以只要条件公平，我不会计较太多。

# 15. 你对加班的看法？

如果工作需要我会义不容辞加班，我现在单身，没有任何家庭负担，可以全身心的投入工作。但同时我也会提高工作效率，减少不必要的加班。

# 16. 你觉得这个工作有什么困难？

工作中出现一些困难是正常的，也是难免的，但是只要有专业的团队、良好的合作精神以及事前周密而充分的准备，任何困难都是可以克服。

# 17. 你希望和什么样的上级共事？

做为刚步入社会的新人，我应该多要求自己尽快熟悉环境、适应环境，而不应该对环境提出什么要求，只要能发挥我的专长就可以了。

# 18. 你工作中难以和同事相处怎么办？

(1)我会服从领导的指挥，配合同事的工作。 (2)我会从自身找原因，仔细分析是不是自己工作做得不够好让领导不满意。 (3)还要看看是不是为人处世方面做得不够好，如果是这样的话我会努力改正。 (4)如果我找不到原因，我会找机会跟他们沟通，请他们指出我的不足，有问题就及时改正。 (5)应该时刻以大局为重，即使在一段时间内，领导和同事对我不理解，我也会做好本职工作，虚心向他们学习，我相信，他们会看见我在努力，总有一天会对我微笑的。

# 19. 你希望从工作中得到最大回报是什么？

对我而言，最重要的是工作能发挥我的特长，能有成就感。同时也期望工作能让我超越现在的水平，不断的提高自己。

# 21. 你为什么认为自己能胜任这个工作？

作为应届毕业生，在工作经验方面的确会有所欠缺，因此在读书期间我一直利用各种机会，在这个行业里做一些兼职/实习。我也发现，实际工作远比书本知识丰富、复杂。但我有较强的责任心、适应能力和学习能力，而且比较勤奋，所以在兼职/实习中均能圆满完成各项工作，从中获取的经验也令我受益非浅。请贵公司放心，学校所学及实习经验使我一定能胜任这个职位。

# 22. 如果录用了你，你会怎么开始工作？

首先听取领导的指示和要求，然后就有关情况进行了解和熟悉，接下来就制定一份近期的工作计划并报给领导批准，最后根据计划开展工作。

# 23. 你还有什么问题？（反问环节）

1.HR 面：

  • 贵公司对新入职的员工有没有什么系统培训？
  • 贵公司的晋升机制是什么样的？
  • 公司的公司氛围、团队建设是怎样子的？
  • 这个岗位出差、加班多吗？
  • 公司有餐补、房补、交通补助之类的吗？
  • 您认为考核这个岗位员工的最重要指标有哪些？
  • 您觉得这个团队的氛围怎么样？

2.技术面：

  • 具体的工作内容是什么？
  • 技术栈？
  • 对校招生的培养，有没有导师？
  • 自己所应聘的部门有多少人？
  • 这次面试多久可以出结果？是否还有下一轮？
  • 自己的表现如何？

# 24. 期望薪资多少？

薪资结构是什么样的？除了年终奖以外是否还有绩效奖？项目奖？公司是否还有其他福利？比如房补、餐补、车补、电话补贴等等。

五险一金的缴纳比例是多少？有没有企业年金？商业补充保险？

一年具体多少薪？试用期多久？试用期期间所有的待遇都是正常的吗？每个月几号发工资？

我了解到公司这个职位薪资是xx～xx，但我在这方面有较多经验，所以我希望可以拿到xxxx。同时我也相信自己完全有能力胜任这份工作，能为公司创造更多的价值。
